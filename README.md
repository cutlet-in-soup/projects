Проекты
1. Artillery Fire Calculator

Десктопное приложение, разработанное на C# WinForms для автоматизации расчётов артиллерийской стрельбы в игре War Thunder.

Основные возможности:

ввод исходных данных;
автоматическое выполнение расчётов;
отображение результатов в графическом интерфейсе;

Исходный код проекта находится в папке ArtilleryFireCalculator.
Document - содержит скрины работы и пояснеиня для приложения
